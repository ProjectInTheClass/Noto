NOTO

팀 활동을 위한 효율적인 일정 공유와 알림 기능을 제공하는 앱


DemoVideo

데모 영상으로 NOTO의 기능을 확인해보세요.

메인 페이지

- 리퀘스트 및 알림의 개수
- 오늘의 할 일
- 사용자가 속한 프로젝트 별 진행 현황
- 모두 보기를 클릭해 추가 정보

프로젝트 페이지(1)


- 리퀘스트 및 알림의 개수
- 프로젝트의 상세 정보
- 캘린더를 통해 프로젝트 내부의 일정

프로젝트 페이지(2)


- 캘린더를 통해 특정 날짜의 일정
- 각 일정을 클릭하여 일정의 상세 정보 페이지로

일정 상세정보 페이지


- 일정에 대한 상세 정보를 확인할 수 있습니다.
- 아래 '일정 URL로 이동' 버튼을 클릭하여, 일정에 대한 URL로 이동할 수 있습니다.

리퀘스트 목록 페이지


- 상단의 '+' 버튼으로 리퀘스트 추가
- 보낸 리퀘스트와 받은 리퀘스트를 확인
- 모두 보기를 클릭해 추가 정보 확인

리퀘스트 상세 페이지


- 리퀘스트에 대한 상세 정보 확인
- 보낸 리퀘스트의 경우, 반려 사유 확인

설정 페이지


- 유저 정보 확인
- 간단한 설정 가능
